## CSS 类

### `containerClass`

- **类型:** `string`
- **默认值:** `'videoshort-container'`
- **描述:** 应用于主容器的 CSS 类。

```js
new VideoShorts('#container', videos, {
  containerClass: 'my-custom-container'
});
```

### `wrapperClass`

- **类型:** `string`
- **默认值:** `'videoshort-wrapper'`
- **描述:** 应用于视频包装器的 CSS 类。

```js
new VideoShorts('#container', videos, {
  wrapperClass: 'my-custom-wrapper'
});
```

### `itemClass`

- **类型:** `string`
- **默认值:** `'videoshort-item'`
- **描述:** 应用于每个视频项的 CSS 类。

```js
new VideoShorts('#container', videos, {
  itemClass: 'my-video-item'
});
```

### `skeletonClass`

- **类型:** `string`
- **默认值:** `'videoshort-skeleton'`
- **描述:** 应用于加载骨架屏的 CSS 类。

```js
new VideoShorts('#container', videos, {
  skeletonClass: 'my-skeleton'
});
```

### `overlayClass`

- **类型:** `string`
- **默认值:** `'videoshort-overlay'`
- **描述:** 应用于覆盖层元素的 CSS 类。

```js
new VideoShorts('#container', videos, {
  overlayClass: 'my-overlay'
});
```

### `playButtonClass`

- **类型:** `string`
- **默认值:** `'videoshort-play-button'`
- **描述:** 应用于播放/暂停按钮的 CSS 类。

```js
new VideoShorts('#container', videos, {
  playButtonClass: 'my-play-btn'
});
```

### `muteButtonClass`

- **类型:** `string`
- **默认值:** `'videoshort-mute-button'`
- **描述:** 应用于静音/取消静音按钮的 CSS 类。

```js
new VideoShorts('#container', videos, {
  muteButtonClass: 'my-mute-btn'
});
```

new VideoShorts('#container', videos, {

new VideoShorts('#container', videos, {

new VideoShorts('#container', videos, {

## 缩略图

### `showThumbnail`

- **类型:** `boolean`
- **默认值:** `true`
- **描述:** 在播放前显示视频缩略图覆盖在播放器上。

```js
new VideoShorts('#container', videos, {
  showThumbnail: true
});
```

### `thumbnailClass`

- **类型:** `string`
- **默认值:** `'videoshort-thumbnail'`
- **描述:** 应用于缩略图元素的 CSS 类。

```js
new VideoShorts('#container', videos, {
  thumbnailClass: 'my-thumbnail'
});
```

### `thumbnailTransitionDuration`

- **类型:** `string`
- **默认值:** `'0.3s'`
- **描述:** 缩略图淡入淡出过渡的持续时间。

```js
new VideoShorts('#container', videos, {
  thumbnailTransitionDuration: '0.5s'
});
```

### `showThumbnailOnPause`

- **类型:** `boolean`
- **默认值:** `false`
- **描述:** 视频暂停时显示缩略图。

```js
new VideoShorts('#container', videos, {
  showThumbnailOnPause: true
});
```

### `thumbnailFallbackIcon`

- **类型:** `string`
- **默认值:** `'<svg>...</svg>'`
- **描述:** 缩略图加载失败时显示的 SVG 图标。

```js
new VideoShorts('#container', videos, {
  thumbnailFallbackIcon: '<svg>...</svg>'
});
```

## 懒加载

### `lazy`

- **类型:** `boolean`
- **默认值:** `true`
- **描述:** 启用或禁用视频的懒加载。

<Tabs>
  <TabItem label="启用">
    ```js
    new VideoShorts('#container', videos, {
      lazy: true // 视频进入视口时加载
    });
    ```
  </TabItem>
  <TabItem label="禁用">
    ```js
    new VideoShorts('#container', videos, {
      lazy: false // 所有视频立即加载
    });
    ```
  </TabItem>
</Tabs>

### `lazyThreshold`

- **类型:** `number`
- **默认值:** `0.1`
- **描述:** Intersection Observer 阈值（0 到 1）。定义元素必须可见多少才加载。

```js
new VideoShorts('#container', videos, {
  lazyThreshold: 0.5 // 元素 50% 可见时加载
});
```

### `lazyRootMargin`

- **类型:** `string`
- **默认值:** `'200px'`
- **描述:** Intersection Observer 根周围的边距。允许在视频进入视口之前加载。

```js
new VideoShorts('#container', videos, {
  lazyRootMargin: '400px' // 进入视口前 400px 开始加载
});
```

## 尺寸

### `width`

- **类型:** `string | null`
- **默认值:** `'360px'`
- **描述:** 每个视频项的宽度。接受有效的 CSS 值。

```js
new VideoShorts('#container', videos, {
  width: '280px' // 用于较小的 Shorts
});
```

```js
new VideoShorts('#container', videos, {
  width: '100%' // 响应式
});
```

### `height`

- **类型:** `string | null`
- **默认值:** `null`
- **描述:** 每个项的固定高度。如果设置，`aspectRatio` 将被忽略。

```js
new VideoShorts('#container', videos, {
  width: '400px',
  height: '225px' // 固定高度
});
```

### `aspectRatio`

- **类型:** `string`
- **默认值:** `'9/16'`
- **描述:** 视频的宽高比。如果设置了 `height` 则忽略。

<Tabs>
  <TabItem label="Shorts (9/16)">
    ```js
    new VideoShorts('#container', videos, {
      aspectRatio: '9/16' // 竖屏格式
    });
    ```
  </TabItem>
  <TabItem label="普通视频 (16/9)">
    ```js
    new VideoShorts('#container', videos, {
      aspectRatio: '16/9' // 宽屏格式
    });
    ```
  </TabItem>
  <TabItem label="正方形 (1/1)">
    ```js
    new VideoShorts('#container', videos, {
      aspectRatio: '1/1' // 正方形格式
    });
    ```
  </TabItem>
</Tabs>

## 播放

### `autoplay`

- **类型:** `boolean`
- **默认值:** `false`
- **描述:** 视频加载后自动开始播放。

```js
new VideoShorts('#container', videos, {
  autoplay: true
});
```

### `muted`

- **类型:** `boolean`
- **默认值:** `false`
- **描述:** 静音播放视频。

```js
new VideoShorts('#container', videos, {
  muted: true,
  autoplay: true // 自动播放通常需要静音
});
```

### `loop`

- **类型:** `boolean`
- **默认值:** `false`
- **描述:** 视频结束后自动重新开始播放。

```js
new VideoShorts('#container', videos, {
  loop: true
});
```

## YouTube 播放器

### `controls`

- **类型:** `number`
- **默认值:** `1`
- **描述:** 显示 YouTube 播放器控件。
  - `0` - 隐藏控件
  - `1` - 显示控件

```js
new VideoShorts('#container', videos, {
  controls: 0 // 不显示控件
});
```

### `rel`

- **类型:** `number`
- **默认值:** `0`
- **描述:** 控制结束时相关视频的显示。
  - `0` - 显示来自同一频道的相关视频
  - `1` - 显示来自任意频道的相关视频

```js
new VideoShorts('#container', videos, {
  rel: 0 // 仅显示同一频道的视频
});
```

### `modestbranding`

- **类型:** `number`
- **默认值:** `1`
- **描述:** 从播放器中移除 YouTube 品牌。
  - `0` - 显示完整品牌
  - `1` - 减少品牌显示

```js
new VideoShorts('#container', videos, {
  modestbranding: 1 // 最小品牌显示
});
```

## 插入位置

### `insertPositionWrapper`

- **类型:** `string`
- **默认值:** `'beforeend'`
- **描述:** 在容器中插入包装器的位置。

| 值 | 描述 |
|-------|-------------|
| `'beforebegin'` | 元素本身之前 |
| `'afterbegin'` | 元素内部，第一个子元素之前 |
| `'beforeend'` | 元素内部，最后一个子元素之后 |
| `'afterend'` | 元素本身之后 |

```js
new VideoShorts('#container', videos, {
  insertPositionWrapper: 'beforeend'
});
```

### `insertPositionItem`

- **类型:** `string`
- **默认值:** `'beforeend'`
- **描述:** 在包装器中插入视频项的位置。

```js
new VideoShorts('#container', videos, {
  insertPositionItem: 'afterbegin'
});
```

使用这些选项可以精确控制视频元素在滑块/轮播结构中的放置位置。

## 回调函数

### `onReady`

- **类型:** `function`
- **描述:** 当播放器准备好播放时调用。

```js
new VideoShorts('#container', videos, {
  onReady: (event, index, instance) => {
    console.log(`视频 ${index} 已准备就绪！`);
  }
});
```

### `onStateChange`

- **类型:** `function`
- **描述:** 当播放器状态改变时调用（播放、暂停、结束等）。

```js
new VideoShorts('#container', videos, {
  onStateChange: (event, index, instance) => {
    if (event.data === YT.PlayerState.PLAYING) {
      console.log(`视频 ${index} 正在播放`);
    }
  }
});
```

### `onPlay`

- **类型:** `function`
- **描述:** 当视频开始播放时调用。

```js
new VideoShorts('#container', videos, {
  onPlay: (event, index, instance) => {
    console.log(`视频 ${index} 开始播放`);
  }
});
```

### `onEnd`

- **类型:** `function`
- **描述:** 当视频结束时调用。

```js
new VideoShorts('#container', videos, {
  onEnd: (event, index, instance) => {
    console.log(`视频 ${index} 已结束`);
    instance.play(index + 1); // 播放下一个视频
  }
});
```

### `onError`

- **类型:** `function`
- **描述:** 当播放器发生错误时调用。

```js
new VideoShorts('#container', videos, {
  onError: (event, index, instance) => {
    console.error(`视频 ${index} 发生错误:`, event.data);
  }
});
```
