## 默认 CSS

VideoShorts 附带一个默认 CSS，您可以通过以下方式引入：

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/videoshots/dist/videoshots.min.css">
```

或通过 import 方式引入：

```js
import 'videoshots/dist/videoshots.min.css';
```

## CSS 结构

### 主容器

```css
.videoshort-container {
  width: 100%;
}
```

### 视频包装器

```css
.videoshort-wrapper {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
  justify-content: center;
  padding: 16px;
}
```

### 视频项

```css
.videoshort-item {
  position: relative;
  overflow: hidden;
  border-radius: 12px;
  background: #1a1a1a;
}
```

### 加载骨架屏

```css
.videoshort-skeleton {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: linear-gradient(
    90deg,
    #2a2a2a 0%,
    #3a3a3a 50%,
    #2a2a2a 100%
  );
  background-size: 200% 100%;
  animation: videoshort-skeleton-loading 1.5s ease-in-out infinite;
}

@keyframes videoshort-skeleton-loading {
  0% {
    background-position: 200% 0;
  }
  100% {
    background-position: -200% 0;
  }
}
```

### Iframe

```css
.videoshort-item iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  border: none;
}
```

## 覆盖样式

### 修改间距

```css
.videoshort-wrapper {
  gap: 24px; /* 默认: 16px */
}
```

### 修改圆角

```css
.videoshort-item {
  border-radius: 20px; /* 默认: 12px */
}
```

### 修改骨架屏背景

```css
.videoshort-skeleton {
  background: linear-gradient(
    90deg,
    #1a1a2e 0%,
    #16213e 50%,
    #1a1a2e 100%
  );
  background-size: 200% 100%;
}
```

## 响应式布局

### 3 列网格布局

```css
.videoshort-item {
  width: calc(33.333% - 11px);
  min-width: 280px;
}

@media (max-width: 768px) {
  .videoshort-item {
    width: calc(50% - 8px);
  }
}

@media (max-width: 480px) {
  .videoshort-item {
    width: 100%;
  }
}
```

### 垂直布局（信息流）

```css
.videoshort-wrapper {
  flex-direction: column;
  align-items: center;
}

.videoshort-item {
  width: 100%;
  max-width: 400px;
}
```

## 自定义类名

您可以定义自己的类名：

```js
new VideoShorts('#container', videos, {
  containerClass: 'my-container',
  wrapperClass: 'my-wrapper',
  itemClass: 'my-item',
  skeletonClass: 'my-skeleton'
});
```

```css
.my-container {
  max-width: 1200px;
  margin: 0 auto;
}

.my-wrapper {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  gap: 20px;
}

.my-item {
  border-radius: 12px;
  overflow: hidden;
  background: #000;
}

.my-skeleton {
  background: linear-gradient(
    90deg,
    #e0e0e0 0%,
    #f0f0f0 50%,
    #e0e0e0 100%
  );
  background-size: 200% 100%;
  animation: skeleton 1.5s ease-in-out infinite;
}

@keyframes skeleton {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

```css
  opacity: 0;
  transform: translateY(20px);

}

.videoshort-item:nth-child(1) { animation-delay: 0.1s; }

.videoshort-item:nth-child(3) { animation-delay: 0.3s; }

```

## 悬停效果

### 悬停缩放

```css
.videoshort-item {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.videoshort-item:hover {
  transform: scale(1.05);
  box-shadow: 0 8px 30px rgba(0, 0, 0, 0.3);
}
```

### 悬停遮罩层

```css
.videoshort-item::after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: rgba(0, 0, 0, 0.5);
  opacity: 0;
  transition: opacity 0.3s;
  pointer-events: none;
}

.videoshort-item:hover::after {
  opacity: 1;
}
```

## 避免样式冲突

使用 `data-videoshort` 属性来隔离您的样式：

```css
[data-videoshort] .videoshort-container {
  /* 您的样式 */
}

[data-videoshort] .videoshort-wrapper {
  /* 您的样式 */
}

[data-videoshort] .videoshort-item {
  /* 您的样式 */
}

[data-videoshort] .videoshort-skeleton {
  /* 您的样式 */
}
```

这可以确保您的样式不会影响页面上的其他元素。

## 重置样式

如果您需要重置默认样式：

```css
.videoshort-container,
.videoshort-wrapper,
.videoshort-item,
.videoshort-skeleton {
  all: unset;
}

.videoshort-container {
  display: block;
}

.videoshort-wrapper {
  display: flex;
  flex-wrap: wrap;
}

.videoshort-item {
  position: relative;
}

.videoshort-item iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
```
