一个轻量级且强大的 JavaScript 库，用于在您的 Web 应用程序中嵌入 YouTube 视频，具有原生懒加载支持和完整的播放控制 API。

<LinkButton href="/zh/doc" variant="primary">文档</LinkButton>

## 特性

<CardGrid stagger>
  <Card title="懒加载" icon="play">
    使用 Intersection Observer API 按需加载视频，以获得更好的性能。
  </Card>
  <Card title="完整 API" icon="settings">
    完全控制播放、音量、静音和其他视频方面。
  </Card>
  <Card title="响应式" icon="device">
    支持 Shorts (9/16) 和常规视频 (16/9) 的自定义纵横比。
  </Card>
  <Card title="零依赖" icon="package">
    独立的库，没有外部依赖。
  </Card>
  <Card title="库友好" icon="plug">
    兼容 SwiperJS、Slick 和其他轮播/滑块库。
  </Card>
</CardGrid>
