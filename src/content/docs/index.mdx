## Features

<CardGrid stagger>
  <Card title="Lazy Loading" icon="play">
    Loads videos only when visible using the Intersection Observer API for optimal performance.
  </Card>
  <Card title="Complete API" icon="settings">
    Full control over playback, volume, mute, seek, and all other video aspects.
  </Card>
  <Card title="Responsive" icon="device">
    Support for custom aspect ratios for Shorts (9/16) and regular videos (16/9).
  </Card>
  <Card title="Zero Dependencies" icon="package">
    Standalone library with no external dependencies. Lightweight and fast.
  </Card>
  <Card title="Library Friendly" icon="plug">
    Compatible with SwiperJS, Slick, and other carousel/slider libraries.
  </Card>
  <Card title="Thumbnail Support" icon="image">
    Show video thumbnails with smooth transitions before playback starts.
  </Card>
</CardGrid>

## Quick Start

```html
<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/eulucastiagolt/videoshorts@1.6.7/dist/videoshorts.min.css">
</head>
<body>
  <div id="videos" data-videoshort></div>

  <script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/videoshorts@1.6.7/dist/videoshorts.min.js"></script>
  <script>
    const player = new VideoShorts('#videos', [
      'https://www.youtube.com/shorts/ZDqo7XROuwM',
      'https://www.youtube.com/watch?v=SGfb6y1j5x0'
    ], {
      lazy: true,
      muted: true,
      loop: true
    });

    player.init();
  </script>
</body>
</html>
```

## Supported URL Formats

VideoShorts works with all YouTube URL formats:

| Format | Example |
|--------|---------|
| YouTube Watch | `https://www.youtube.com/watch?v=VIDEO_ID` |
| YouTube Shorts | `https://www.youtube.com/shorts/VIDEO_ID` |
| YouTube Embed | `https://www.youtube.com/embed/VIDEO_ID` |
| youtu.be | `https://youtu.be/VIDEO_ID` |
