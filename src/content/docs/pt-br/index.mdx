## Início Rápido

```html
<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/eulucastiagolt/videoshorts@1.6.4/dist/videoshots.min.css">
</head>
<body>
  <div id="videos" data-videoshort></div>

  <script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/videoshorts@1.6.4/dist/videoshots.min.js"></script>
  <script>
    const player = new VideoShorts('#videos', [
      'https://www.youtube.com/shorts/ZDqo7XROuwM',
      'https://www.youtube.com/watch?v=SGfb6y1j5x0'
    ], {
      lazy: true,
      muted: true,
      loop: true
    });

    player.init();
  </script>
</body>
</html>
```

## Formatos de URL Suportados

VideoShorts funciona com todos os formatos de URL do YouTube:

| Formato | Exemplo |
|---------|---------|
| YouTube Watch | `https://www.youtube.com/watch?v=VIDEO_ID` |
| YouTube Shorts | `https://www.youtube.com/shorts/VIDEO_ID` |
| YouTube Embed | `https://www.youtube.com/embed/VIDEO_ID` |
| youtu.be | `https://youtu.be/VIDEO_ID` |
