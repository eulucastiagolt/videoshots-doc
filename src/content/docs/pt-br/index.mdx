Uma biblioteca JavaScript leve e poderosa para incorporar vídeos do YouTube em suas aplicações web, com suporte nativo a lazy loading e uma API completa para controle de reprodução.

<LinkButton href="/pt-br/doc" variant="primary">Documentação</LinkButton>

## Recursos

<CardGrid stagger>
  <Card title="Carregamento Sob Demanda" icon="play">
    Carrega vídeos sob demanda usando a API Intersection Observer para melhor desempenho.
  </Card>
  <Card title="API Completa" icon="settings">
    Controle total sobre reprodução, volume, mudo e outros aspectos do vídeo.
  </Card>
  <Card title="Responsivo" icon="device">
    Suporte para proporções personalizadas para Shorts (9/16) e vídeos regulares (16/9).
  </Card>
  <Card title="Zero Dependências" icon="package">
    Biblioteca independente, sem dependências externas.
  </Card>
  <Card title="Compatível com Bibliotecas" icon="plug">
    Compatível com SwiperJS, Slick e outras bibliotecas de carousel/slider.
  </Card>
</CardGrid>
