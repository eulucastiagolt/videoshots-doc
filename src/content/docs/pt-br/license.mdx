## O que isso significa?

A licença MIT é uma licença permissiva que permite:

- **Uso comercial** - Você pode usar o software comercialmente
- **Modificação** - Você pode modificar o código
- **Distribuição** - Você pode distribuir o código
- **Uso privado** - Você pode usar o código privadamente

O único requisito é incluir o aviso de copyright e a licença em todas as cópias.
