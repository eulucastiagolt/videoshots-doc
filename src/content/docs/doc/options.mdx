## CSS Classes

### `containerClass`

- **Type:** `string`
- **Default:** `'videoshort-container'`
- **Description:** CSS class applied to the main container.

```js
new VideoShorts('#container', videos, {
  containerClass: 'my-custom-container'
});
```

### `wrapperClass`

- **Type:** `string`
- **Default:** `'videoshort-wrapper'`
- **Description:** CSS class applied to the video wrapper.

```js
new VideoShorts('#container', videos, {
  wrapperClass: 'my-custom-wrapper'
});
```

### `itemClass`

- **Type:** `string`
- **Default:** `'videoshort-item'`
- **Description:** CSS class applied to each video item.

```js
new VideoShorts('#container', videos, {
  itemClass: 'my-video-item'
});
```

### `skeletonClass`

- **Type:** `string`
- **Default:** `'videoshort-skeleton'`
- **Description:** CSS class applied to the loading skeleton.

```js
new VideoShorts('#container', videos, {
  skeletonClass: 'my-skeleton'
});
```

### `overlayClass`

- **Type:** `string`
- **Default:** `'videoshort-overlay'`
- **Description:** CSS class applied to the overlay element.

```js
new VideoShorts('#container', videos, {
  overlayClass: 'my-overlay'
});
```

### `playButtonClass`

- **Type:** `string`
- **Default:** `'videoshort-play-button'`
- **Description:** CSS class applied to the play/pause button.

```js
new VideoShorts('#container', videos, {
  playButtonClass: 'my-play-btn'
});
```

### `muteButtonClass`

- **Type:** `string`
- **Default:** `'videoshort-mute-button'`
- **Description:** CSS class applied to the mute/unmute button.

```js
new VideoShorts('#container', videos, {
  muteButtonClass: 'my-mute-btn'
});
```

## Thumbnail

### `showThumbnail`

- **Type:** `boolean`
- **Default:** `true`
- **Description:** Shows video thumbnail over the player before playback.

```js
new VideoShorts('#container', videos, {
  showThumbnail: true
});
```

### `thumbnailClass`

- **Type:** `string`
- **Default:** `'videoshort-thumbnail'`
- **Description:** CSS class applied to the thumbnail element.

```js
new VideoShorts('#container', videos, {
  thumbnailClass: 'my-thumbnail'
});
```

### `thumbnailTransitionDuration`

- **Type:** `string`
- **Default:** `'0.3s'`
- **Description:** Duration of the thumbnail fade transition.

```js
new VideoShorts('#container', videos, {
  thumbnailTransitionDuration: '0.5s'
});
```

### `showThumbnailOnPause`

- **Type:** `boolean`
- **Default:** `false`
- **Description:** Shows thumbnail when video is paused.

```js
new VideoShorts('#container', videos, {
  showThumbnailOnPause: true
});
```

### `thumbnailFallbackIcon`

- **Type:** `string`
- **Default:** `'<svg>...</svg>'`
- **Description:** SVG icon displayed when thumbnail fails to load.

```js
new VideoShorts('#container', videos, {
  thumbnailFallbackIcon: '<svg>...</svg>'
});
```

## Lazy Loading

### `lazy`

- **Type:** `boolean`
- **Default:** `true`
- **Description:** Enables or disables lazy loading of videos.

<Tabs>
  <TabItem label="Enabled">
    ```js
    new VideoShorts('#container', videos, {
      lazy: true // videos load when they enter the viewport
    });
    ```
  </TabItem>
  <TabItem label="Disabled">
    ```js
    new VideoShorts('#container', videos, {
      lazy: false // all videos load immediately
    });
    ```
  </TabItem>
</Tabs>

### `lazyThreshold`

- **Type:** `number`
- **Default:** `0.1`
- **Description:** Intersection Observer threshold (0 to 1). Defines how much of the element must be visible before loading.

```js
new VideoShorts('#container', videos, {
  lazyThreshold: 0.5 // loads when 50% of element is visible
});
```

### `lazyRootMargin`

- **Type:** `string`
- **Default:** `'200px'`
- **Description:** Margin around the Intersection Observer root. Allows loading videos before they enter the viewport.

```js
new VideoShorts('#container', videos, {
  lazyRootMargin: '400px' // loads 400px before entering viewport
});
```

## Dimensions

### `width`

- **Type:** `string | null`
- **Default:** `'360px'`
- **Description:** Width of each video item. Accepts valid CSS values.

```js
new VideoShorts('#container', videos, {
  width: '280px' // for smaller Shorts
});
```

```js
new VideoShorts('#container', videos, {
  width: '100%' // responsive
});
```

### `height`

- **Type:** `string | null`
- **Default:** `null`
- **Description:** Fixed height of each item. If set, `aspectRatio` is ignored.

```js
new VideoShorts('#container', videos, {
  width: '400px',
  height: '225px' // fixed height
});
```

### `aspectRatio`

- **Type:** `string`
- **Default:** `'9/16'`
- **Description:** Aspect ratio of the videos. Ignored if `height` is set.

<Tabs>
  <TabItem label="Shorts (9/16)">
    ```js
    new VideoShorts('#container', videos, {
      aspectRatio: '9/16' // vertical format
    });
    ```
  </TabItem>
  <TabItem label="Normal Videos (16/9)">
    ```js
    new VideoShorts('#container', videos, {
      aspectRatio: '16/9' // widescreen format
    });
    ```
  </TabItem>
  <TabItem label="Square (1/1)">
    ```js
    new VideoShorts('#container', videos, {
      aspectRatio: '1/1' // square format
    });
    ```
  </TabItem>
</Tabs>

## Playback

### `autoplay`

- **Type:** `boolean`
- **Default:** `false`
- **Description:** Starts playback automatically when the video loads.

```js
new VideoShorts('#container', videos, {
  autoplay: true
});
```

### `muted`

- **Type:** `boolean`
- **Default:** `false`
- **Description:** Starts the video without audio.

```js
new VideoShorts('#container', videos, {
  muted: true,
  autoplay: true // autoplay usually requires muted
});
```

### `loop`

- **Type:** `boolean`
- **Default:** `false`
- **Description:** Restarts the video automatically when it ends.

```js
new VideoShorts('#container', videos, {
  loop: true
});
```

## YouTube Player

### `controls`

- **Type:** `number`
- **Default:** `1`
- **Description:** Displays YouTube player controls.
  - `0` - Hides controls
  - `1` - Shows controls

```js
new VideoShorts('#container', videos, {
  controls: 0 // no visible controls
});
```

### `rel`

- **Type:** `number`
- **Default:** `0`
- **Description:** Controls the display of related videos at the end.
  - `0` - Shows related videos from the same channel
  - `1` - Shows related videos from any channel

```js
new VideoShorts('#container', videos, {
  rel: 0 // only videos from the same channel
});
```

### `modestbranding`

- **Type:** `number`
- **Default:** `1`
- **Description:** Removes YouTube branding from the player.
  - `0` - Shows full branding
  - `1` - Reduces branding

```js
new VideoShorts('#container', videos, {
  modestbranding: 1 // minimal branding
});
```

## Insert Position

### `insertPositionWrapper`

- **Type:** `string`
- **Default:** `'beforeend'`
- **Description:** Position to insert wrapper element in container.

| Value | Description |
|-------|-------------|
| `'beforebegin'` | Before the element itself |
| `'afterbegin'` | Just inside the element, before its first child |
| `'beforeend'` | Just inside the element, after its last child |
| `'afterend'` | After the element itself |

```js
new VideoShorts('#container', videos, {
  insertPositionWrapper: 'beforeend'
});
```

### `insertPositionItem`

- **Type:** `string`
- **Default:** `'beforeend'`
- **Description:** Position to insert video items in wrapper.

```js
new VideoShorts('#container', videos, {
  insertPositionItem: 'afterbegin'
});
```

Use these options to control the exact placement of video elements within your slider/carousel structure.

## Callbacks

### `onReady`

- **Type:** `function`
- **Description:** Called when a player is ready for playback.

```js
new VideoShorts('#container', videos, {
  onReady: (event, index, instance) => {
    console.log(`Video ${index} ready!`);
  }
});
```

### `onStateChange`

- **Type:** `function`
- **Description:** Called when the player state changes (play, pause, end, etc).

```js
new VideoShorts('#container', videos, {
  onStateChange: (event, index, instance) => {
    if (event.data === YT.PlayerState.PLAYING) {
      console.log(`Video ${index} is playing`);
    }
  }
});
```

### `onPlay`

- **Type:** `function`
- **Description:** Called when a video starts playing.

```js
new VideoShorts('#container', videos, {
  onPlay: (event, index, instance) => {
    console.log(`Video ${index} started playing`);
  }
});
```

### `onEnd`

- **Type:** `function`
- **Description:** Called when a video ends.

```js
new VideoShorts('#container', videos, {
  onEnd: (event, index, instance) => {
    console.log(`Video ${index} ended`);
    instance.play(index + 1); // play next video
  }
});
```

### `onError`

- **Type:** `function`
- **Description:** Called when a player error occurs.

```js
new VideoShorts('#container', videos, {
  onError: (event, index, instance) => {
    console.error(`Error on video ${index}:`, event.data);
  }
});
```
