VideoShorts is a lightweight and powerful JavaScript library for embedding YouTube videos in your web applications, with native lazy loading support and a complete API for playback control.

## Features

<CardGrid stagger>
  <Card title="Lazy Loading" icon="play">
    Loads videos on demand using the Intersection Observer API for better performance.
  </Card>
  <Card title="Complete API" icon="settings">
    Full control over playback, volume, mute, and other video aspects.
  </Card>
  <Card title="Responsive" icon="device">
    Support for custom aspect ratios for Shorts (9/16) and regular videos (16/9).
  </Card>
  <Card title="Zero Dependencies" icon="package">
    Standalone library with no external dependencies.
  </Card>
</CardGrid>

## Why VideoShorts?

- **Performance**: Automatic lazy loading reduces initial page load time
- **Flexibility**: Support for multiple YouTube URL formats (watch, shorts, embed, youtu.be)
- **Full Control**: Intuitive API to control all aspects of playback
- **Customization**: Customizable CSS and extensible configuration options
- **Skeleton Loading**: Smooth loading experience with skeleton animation

## Quick Start

```html
<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/eulucastiagolt/videoshorts@1.0.0/dist/videoshots.min.css">
</head>
<body>
  <div id="videos" data-videoshort></div>

  <script src="https://cdn.jsdelivr.net/gh/eulucastiagolt/videoshorts@1.0.0/dist/videoshots.min.js"></script>
  <script>
    const videos = [
      'https://www.youtube.com/shorts/ZDqo7XROuwM',
      'https://www.youtube.com/watch?v=SGfb6y1j5x0'
    ];

    const player = new VideoShorts('#videos', videos);
    player.init();
  </script>
</body>
</html>
```

## Shorts Support

VideoShorts is designed with native support for YouTube Shorts, allowing you to display vertical format videos (9/16) elegantly in your applications.

## Browser Support

- Chrome 51+
- Firefox 55+
- Safari 12.1+
- Edge 15+

Requires `IntersectionObserver` API (lazy loading).
