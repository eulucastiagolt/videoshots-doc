## Default CSS

VideoShorts comes with a default CSS that you can include:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/videoshorts/dist/videoshorts.min.css">
```

Or via import:

```js
import 'videoshorts/dist/videoshorts.min.css';
```

## CSS Structure

### Main Container

```css
.videoshort-container {
  width: 100%;
}
```

### Video Wrapper

```css
.videoshort-wrapper {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
  justify-content: center;
  padding: 16px;
}
```

### Video Item

```css
.videoshort-item {
  position: relative;
  overflow: hidden;
  border-radius: 12px;
  background: #1a1a1a;
}
```

### Loading Skeleton

```css
.videoshort-skeleton {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: linear-gradient(
    90deg,
    #2a2a2a 0%,
    #3a3a3a 50%,
    #2a2a2a 100%
  );
  background-size: 200% 100%;
  animation: videoshort-skeleton-loading 1.5s ease-in-out infinite;
}

@keyframes videoshort-skeleton-loading {
  0% {
    background-position: 200% 0;
  }
  100% {
    background-position: -200% 0;
  }
}
```

### Iframe

```css
.videoshort-item iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  border: none;
}
```

## Overriding Styles

### Change Gap

```css
.videoshort-wrapper {
  gap: 24px; /* default: 16px */
}
```

### Change Border Radius

```css
.videoshort-item {
  border-radius: 20px; /* default: 12px */
}
```

### Change Skeleton Background

```css
.videoshort-skeleton {
  background: linear-gradient(
    90deg,
    #1a1a2e 0%,
    #16213e 50%,
    #1a1a2e 100%
  );
  background-size: 200% 100%;
}
```

## Responsive Layout

### 3 Column Grid

```css
.videoshort-item {
  width: calc(33.333% - 11px);
  min-width: 280px;
}

@media (max-width: 768px) {
  .videoshort-item {
    width: calc(50% - 8px);
  }
}

@media (max-width: 480px) {
  .videoshort-item {
    width: 100%;
  }
}
```

### Vertical Layout (Feed)

```css
.videoshort-wrapper {
  flex-direction: column;
  align-items: center;
}

.videoshort-item {
  width: 100%;
  max-width: 400px;
}
```

## Custom Classes

You can define your own classes:

```js
new VideoShorts('#container', videos, {
  containerClass: 'my-container',
  wrapperClass: 'my-wrapper',
  itemClass: 'my-item',
  skeletonClass: 'my-skeleton'
});
```

```css
.my-container {
  max-width: 1200px;
  margin: 0 auto;
}

.my-wrapper {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  gap: 20px;
}

.my-item {
  border-radius: 12px;
  overflow: hidden;
  background: #000;
}

.my-skeleton {
  background: linear-gradient(
    90deg,
    #e0e0e0 0%,
    #f0f0f0 50%,
    #e0e0e0 100%
  );
  background-size: 200% 100%;
  animation: skeleton 1.5s ease-in-out infinite;
}

@keyframes skeleton {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

## Entry Animation

```css
.videoshort-item {
  opacity: 0;
  transform: translateY(20px);
  animation: fadeInUp 0.5s ease forwards;
}

.videoshort-item:nth-child(1) { animation-delay: 0.1s; }
.videoshort-item:nth-child(2) { animation-delay: 0.2s; }
.videoshort-item:nth-child(3) { animation-delay: 0.3s; }
.videoshort-item:nth-child(4) { animation-delay: 0.4s; }
.videoshort-item:nth-child(5) { animation-delay: 0.5s; }
```

## Hover Effects

### Scale on Hover

```css
.videoshort-item {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.videoshort-item:hover {
  transform: scale(1.05);
  box-shadow: 0 8px 30px rgba(0, 0, 0, 0.3);
}
```

### Overlay on Hover

```css
.videoshort-item::after {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: rgba(0, 0, 0, 0.5);
  opacity: 0;
  transition: opacity 0.3s;
  pointer-events: none;
}

.videoshort-item:hover::after {
  opacity: 1;
}
```

## Avoiding Conflicts

Use the `data-videoshort` attribute to isolate your styles:

```css
[data-videoshort] .videoshort-container {
  /* your styles */
}

[data-videoshort] .videoshort-wrapper {
  /* your styles */
}

[data-videoshort] .videoshort-item {
  /* your styles */
}

[data-videoshort] .videoshort-skeleton {
  /* your styles */
}
```

This ensures your styles don't affect other elements on the page.

## Reset Styles

If you need to reset the default styles:

```css
.videoshort-container,
.videoshort-wrapper,
.videoshort-item,
.videoshort-skeleton {
  all: unset;
}

.videoshort-container {
  display: block;
}

.videoshort-wrapper {
  display: flex;
  flex-wrap: wrap;
}

.videoshort-item {
  position: relative;
}

.videoshort-item iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
```
