A lightweight and powerful JavaScript library for embedding YouTube videos in your web applications, with native lazy loading support and a complete API for playback control.

<LinkButton href="/en/doc" variant="primary">Documentation</LinkButton>

## Features

<CardGrid stagger>
  <Card title="Lazy Loading" icon="play">
    Loads videos on demand using the Intersection Observer API for better performance.
  </Card>
  <Card title="Complete API" icon="settings">
    Full control over playback, volume, mute, and other video aspects.
  </Card>
  <Card title="Responsive" icon="device">
    Support for custom aspect ratios for Shorts (9/16) and regular videos (16/9).
  </Card>
  <Card title="Zero Dependencies" icon="package">
    Standalone library with no external dependencies.
  </Card>
</CardGrid>
