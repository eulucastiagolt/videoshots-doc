title: Events
description: Event callbacks available in VideoShorts.

Events are defined through callbacks in the configuration options.

## onReady

Called when a YouTube player is fully loaded and ready for playback.

- **Parameters:**
  - `event`: YouTube Player API event
  - `index`: Video index
  - `instance`: VideoShorts instance

```js
new VideoShorts('#container', videos, {
  onReady: (event, index, instance) => {
    console.log(`Video ${index} is ready!`);
    console.log('Duration:', instance.getDuration(index));
  }
});
```

## onStateChange

Called when the player state changes.

- **Parameters:**
  - `event`: Event with `data` containing the new state
  - `index`: Video index
  - `instance`: VideoShorts instance

### Player States

| Value | Constant | Description |
|-------|----------|-------------|
| -1 | `YT.PlayerState.UNSTARTED` | Unstarted |
| 0 | `YT.PlayerState.ENDED` | Ended |
| 1 | `YT.PlayerState.PLAYING` | Playing |
| 2 | `YT.PlayerState.PAUSED` | Paused |
| 3 | `YT.PlayerState.BUFFERING` | Buffering |
| 5 | `YT.PlayerState.CUED` | Video cued |

```js
new VideoShorts('#container', videos, {
  onStateChange: (event, index, instance) => {
    switch (event.data) {
      case YT.PlayerState.PLAYING:
        console.log(`Video ${index} started playing`);
        break;
      case YT.PlayerState.PAUSED:
        console.log(`Video ${index} was paused`);
        break;
      case YT.PlayerState.ENDED:
        console.log(`Video ${index} ended`);
        instance.play(index + 1); // play next video
        break;
    }
  }
});
```

## onError

Called when a player error occurs.

- **Parameters:**
  - `event`: Event with `data` containing the error code
  - `index`: Video index
  - `instance`: VideoShorts instance

### Error Codes

| Code | Description |
|------|-------------|
| 2 | Invalid request |
| 5 | HTML5 player error |
| 100 | Video not found |
| 101 | Playback not allowed by owner |
| 150 | Playback not allowed by owner |

```js
new VideoShorts('#container', videos, {
  onError: (event, index, instance) => {
    console.error(`Error on video ${index}:`, event.data);
    
    switch (event.data) {
      case 100:
        console.error('Video not found or removed');
        break;
      case 101:
      case 150:
        console.error('Playback not allowed by owner');
        break;
    }
  }
});
```

## Complete Example

```js
const videos = new VideoShorts('#container', videoUrls, {
  onReady: (event, index, instance) => {
    console.log(`Video ${index} ready`);
    updateUI();
  },
  
  onStateChange: (event, index, instance) => {
    if (event.data === YT.PlayerState.PLAYING) {
      pauseOtherVideos(index);
    }
    if (event.data === YT.PlayerState.ENDED) {
      playNextVideo(index);
    }
  },
  
  onError: (event, index, instance) => {
    showErrorMessage(index, event.data);
  }
});

await videos.init();
```

## Framework Integration

### React

```jsx
import { useEffect, useRef } from 'react';
import VideoShorts from 'videoshots';
import 'videoshots/dist/videoshots.min.css';

function VideoPlayer({ videos }) {
  const containerRef = useRef(null);
  const instanceRef = useRef(null);

  useEffect(() => {
    instanceRef.current = new VideoShorts(containerRef.current, videos, {
      onReady: (event, index) => {
        console.log('Ready:', index);
      },
      onStateChange: (event, index) => {
        console.log('State changed:', index, event.data);
      }
    });

    instanceRef.current.init();

    return () => {
      instanceRef.current?.destroy();
    };
  }, [videos]);

  return <div ref={containerRef} data-videoshort />;
}
```

### Vue

```vue
<template>
  <div ref="container" data-videoshort></div>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue';
import VideoShorts from 'videoshots';
import 'videoshots/dist/videoshots.min.css';

const props = defineProps(['videos']);
const container = ref(null);
let instance = null;

onMounted(async () => {
  instance = new VideoShorts(container.value, props.videos, {
    onReady: (event, index) => {
      console.log('Ready:', index);
    }
  });
  await instance.init();
});

onUnmounted(() => {
  instance?.destroy();
});
</script>
```
